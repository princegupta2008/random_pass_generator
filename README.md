# Random Password Generator

## Description
This project generates random passwords and includes both a command-line and GUI version.

## Requirements
- Required Libraries: `pyperclip`, `tkinter` (may require `python-tk` on Linux)
- Install Command:
  ```bash
  pip install pyperclip tk
  
# Usage
`random_pass.py`: Run this for the command-line password generator.\
`random_pass_gui.py`: Run this for the GUI version.
